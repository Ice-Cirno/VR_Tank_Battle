# VR Tank Project





## Prefab 说明

### Google VR部分

以下4个Prefab导入自Google VR SDK for Unity

- GvrControllerMain
- GvrEventSystem
- GvrEditorEmulator
- GvrInstantPreviewMain

### LeapMotion 部分

- Leap Rig：导入自Leap Motion SDK Core

### Game 部分

- LevelArt：场景地图
- Tank：玩家操控单位
- Player Camera：主摄像机
- PostFX：图像后处理
